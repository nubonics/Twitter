A twitter app is required to use this program.
For more details, please see: https://developer.twitter.com/en/apps

# To run the program
Fill in the credentials.json file with your twitter app credentials
## For tweets that are less than a week old, leave the file as is with 8 in Line 55 column 44
Otherwise modify the file tweet_dumperv1.py Line 55 Column 44 and replace the 8 with a 31
python main.py
